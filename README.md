# Greenearth Organic - Invoice Manager

![Greenearth](https://img.shields.io/badge/Greenearth%20Organic-Portal-success?style=for-the-badge) ![Offline](https://img.shields.io/badge/Offline-Supported-blue?style=for-the-badge) ![No%20DB](https://img.shields.io/badge/Local-Storage-orange?style=for-the-badge)

A secure, premium, frontend-only Web Application designed specifically to generate tax-compliant invoices and share them effortlessly via PDF or WhatsApp.

## 🚀 Features

*   🔐 **Secure Owner Portal:** Direct gating so only the business owner can manage data. (Default credentials: `admin` / `1234`).
*   📡 **100% Offline Capable & Free Database:** Because the data is stored using the browser's `localStorage`, you do not need active internet just to write or view old invoices! Data is stored directly on your phone/computer.
*   📱 **Mobile Optimised & App-like:** Can be saved to your phone's Home Screen and runs exactly like a native app.
*   🧾 **One-Click WhatsApp / PDF Sharing:** Dynamic integrations to talk directly to users without downloading anything.
*   ⚙️ **Product Management Module:** Add or delete the products dynamically via the robust dashboard.

## 💻 Tech Stack
*   **HTML5**
*   **TailwindCSS (via CDN):** Rapid and stunning styling.
*   **Vanilla JS (ES6):** No messy frameworks. All logic is embedded.
*   **html2pdf.js:** Client-side PDF rendering.
*   **EmailJS:** Pre-wired configurations for sending browser-based emails.

## 🌐 Live Deployment
Deployed automatically to [Netlify App](https://greenearth-invoice-app-2026.netlify.app).

## 📱 How to Use on Mobile (Offline Mode)
1. Open the Live Site URL in your phone (Chrome/Safari).
2. Tap the browser Menu (three dots) -> click **"Add to Home Screen"**.
3. Now the Invoice Generator will be installed directly on your home screen. You can use it even when your internet is off!

## ⚙️ Disclaimer
This project uses `LocalStorage`. Clearing your browser cache/storage may erase data unless backed up.
